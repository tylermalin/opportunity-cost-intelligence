# Contributing to Opportunity Cost Intelligence

OCI is an early open experiment.

The goal is to help AI systems better protect human time, attention, and agency.

## Useful Contributions

- New Skill Files
- Claude, ChatGPT, Cursor, Windsurf, and agent integrations
- Better time-value heuristics
- Browser extension prototypes
- Examples from real workflows
- Research links
- Critiques of the model
- Professional boundary improvements
- Lightweight tracking tools

## Contribution Standard

Keep it simple.

OCI should not become another complex productivity system.

Every contribution should answer:

1. Does this reduce cognitive burden?
2. Does this make hidden labor more visible?
3. Does this help someone decide whether to continue, stop, delegate, automate, or seek help?
4. Is this easy to fork?

## Pull Request Format

```text
What changed:
Why it matters:
How it reduces hidden labor:
Any risks or limitations:
```

## Design Constraint

If the tool takes more attention than it returns, it is failing.
