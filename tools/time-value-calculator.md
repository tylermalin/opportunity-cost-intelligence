# Time Value Calculator

This is a rough tool for making invisible time costs visible.

## Basic Formula

```text
Personal Time Value = annual income target / 1,500 focused hours
```

Why 1,500?

A typical work year has about 2,000 nominal hours. Focused, high-quality, decision-relevant hours are usually much lower after meetings, admin, interruptions, logistics, and recovery.

## Fast Defaults

Use these when a user does not know their number:

```text
$25/hr = constrained budget or early-career baseline
$50/hr = professional baseline
$100/hr = founder/operator baseline
$250/hr = executive/strategic time
```

## Task Cost Formula

```text
Task Cost = active minutes / 60 * personal time value
```

Example:

```text
90 minutes * $100/hr = $150 time cost
```

## Decision Rule

A task may be worth doing yourself when:

```text
money saved + learning value + control value > time cost + stress cost + error risk
```

## Quick Scoring

```text

```

Then compare:

```text
DIY Value = money saved + learning value + control value
DIY Burden = time cost + stress cost + error risk
```

If DIY Burden is higher, consider delegating, automating, simplifying, or stopping.

## Important Caveat

This is not a perfect economic model.

It is a prompt for honest reflection.
